# MULTIVON

**Multivon** is a modern **MERN stack** application designed to showcase projects, clients, and provide consultation features with a user-friendly interface.

---

## TECHNOLOGIES USED

- **MongoDB** – Database
- **Express.js** – Backend framework
- **React.js** – Frontend library
- **Node.js** – Runtime environment
- **Axios** – API calls
- **Vercel** – Frontend deployment
- **Render** – Backend deployment

---

## FEATURES

- **Responsive frontend** built with React
- **Admin Dashboard** for managing projects, clients, and content
- **Protected Routes** for secure access
- **Contact Form** and **Consultation Form**
- **Project and Client Sections** with images and details
- **Authentication System** with Login

## INSTALLATION

## 1. Clone the repository:
git clone https://github.com/Leeni-Patidar/multivon.git

## 2.Backend Setup
cd Backend
npm install
node server.js

## 3.Frontend Setup
cd Frontend
npm install
npm run dev
